## 2. String Harmonics

#### **1. Identify the Given Variables**
First, we need to make sure our units are standard (converting centimeters to meters).
* Length ($L$) = 64 cm = 0.64 m
* Fundamental frequency ($f_1$) = 330 Hz

#### **2. Find the Wavelength ($\lambda$)**
For a string fixed at both ends, the fundamental frequency (which has exactly one antinode) means that only half of a full wave fits on the string. 
$$L = \frac{\lambda}{2}$$

To find the full wavelength, we rearrange the formula to multiply the length by 2:
$$\lambda = 2L$$
$$\lambda = 2 \cdot 0.64$$
$$\lambda = 1.28 \text{ m}$$

#### **3. Calculate the Wave Speed ($v$)**
Now that we have the frequency and the wavelength, we use the universal wave equation to find the speed:
$$v = f \cdot \lambda$$
$$v = 330 \cdot 1.28$$
**$v = 422.4 \text{ m/s}$**
