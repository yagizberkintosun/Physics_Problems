# Physics Problem: Electromagnetic Spectrum Ordering

## Problem Statement
List the following types of electromagnetic radiation in order of **increasing wavelength**:
*   Infrared
*   Ultraviolet
*   Microwaves
*   X-rays
*   Radio waves
*   Gamma rays

---

## 1. Physical Principles
Electromagnetic radiation is classified by its wavelength ($\lambda$) and frequency ($f$). These two properties are related to the speed of light ($c$) by the following equation:
$$c = \lambda \cdot f$$

Because the speed of light is constant in a vacuum, wavelength and frequency are **inversely proportional**. This means:
*   **High Frequency/High Energy** radiation has the **shortest wavelengths**.
*   **Low Frequency/Low Energy** radiation has the **longest wavelengths**.

## 2. Comparative Analysis
To order these correctly, we analyze their positions in the spectrum based on their energy levels:

| Radiation Type | Typical Wavelength Range | Context/Energy |
| :--- | :--- | :--- |
| **Gamma rays** | $< 10^{-11} \text{ m}$ | Highest energy; shortest $\lambda$ |
| **X-rays** | $10^{-11} \text{ to } 10^{-8} \text{ m}$ | High energy; used in medical imaging |
| **Ultraviolet** | $10 \text{ nm to } 400 \text{ nm}$ | Just above the visible spectrum |
| **Infrared** | $700 \text{ nm to } 1 \text{ mm}$ | Just below the visible spectrum (heat) |
| **Microwaves** | $1 \text{ mm to } 1 \text{ m}$ | Low energy; used in radar and cooking |
| **Radio waves** | $> 1 \text{ m}$ | Lowest energy; longest $\lambda$ |

## 3. Step-by-Step Ordering
To list them in order of **increasing wavelength**, we start with the type that has the highest frequency (Gamma rays) and end with the type that has the lowest frequency (Radio waves).

1.  **Gamma rays** (Shortest wavelength)
2.  **X-rays**
3.  **Ultraviolet**
4.  **Infrared**
5.  **Microwaves**
6.  **Radio waves** (Longest wavelength)

## 4. Final Result
The correct order of increasing wavelength for the provided types of radiation is:

$$\mathbf{Gamma\ rays < X-rays < Ultraviolet < Infrared < Microwaves < Radio\ waves}$$
