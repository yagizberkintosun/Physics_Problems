## 6. Function Analysis

Find the local maxima or minima for the function:
$$f(x) = 3x^2 - 12x + 7$$

---

### Step 1: Find the first derivative $f'(x)$
The derivative represents the slope of the function. To find critical points, we differentiate with respect to $x$:
$$f'(x) = \frac{d}{dx}(3x^2 - 12x + 7)$$
$$f'(x) = 6x - 12$$

### Step 2: Set the derivative to zero
The local maximum or minimum occurs where the slope is zero:
$$6x - 12 = 0$$
$$6x = 12$$
$$\mathbf{x = 2}$$

### Step 3: Determine if it is a Maximum or Minimum
We use the **Second Derivative Test**:
$$f''(x) = \frac{d}{dx}(6x - 12) = 6$$

Since $f''(x) > 0$ (positive), the parabola opens upward, meaning the point is a **local minimum**.



### Step 4: Calculate the coordinates of the vertex
Substitute $x = 2$ back into the original function $f(x)$:
$$f(2) = 3(2)^2 - 12(2) + 7$$
$$f(2) = 3(4) - 24 + 7$$
$$f(2) = 12 - 24 + 7 = -5$$

---

### Conclusion
The function has a **local minimum** at the point:
$$\mathbf{(2, -5)}$$
